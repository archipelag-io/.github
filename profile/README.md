### What is Archipelag.io?

Archipelag.io connects people who need AI compute with people who have idle GPUs. Consumers submit inference jobs — LLM chat, image generation — and the platform routes them to the nearest available Island (compute contributor), streaming results back in real time. Islands earn credits for every job they complete.

The API is OpenAI-compatible. Switch one URL and your existing code works.

**Currently in [open beta](https://archipelag.io/blog/open-beta-announcement/) through June 2026.** Free credits for new accounts.

### For developers

- **OpenAI-compatible API** — drop-in replacement, one URL change
- **Python SDK** — `pip install archipelag` → [`archipelagio-python`](https://github.com/archipelag-io/archipelagio-python)
- **JavaScript SDK** — `npm install @archipelag/sdk` → [`archipelagio-js`](https://github.com/archipelag-io/archipelagio-js)
- **CLI** — `island chat "Hello"` → [`island-cli`](https://github.com/archipelag-io/island-cli)
- **API reference** — interactive docs at [api.archipelag.io](https://api.archipelag.io)

### For GPU owners

Contribute your idle hardware and earn credits. You set the price, the platform matches you with nearby demand. Sandboxed execution means no arbitrary code ever touches your machine — only network-approved, signed containers.

[Learn more →](https://archipelag.io/earn)
